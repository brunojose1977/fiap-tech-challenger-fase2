O código que representa o fluxograma do processo de integração do seu modelo com o Azure ML, cobrindo desde o código local até o Deployment.

Como usar este código no draw.io:

Copie o código XML abaixo.

Vá para app.diagrams.net.

https://app.diagrams.net/

Clique em File (Arquivo) > Import from (Importar de) > Text... (Texto...).

Cole o código e clique em Import.
